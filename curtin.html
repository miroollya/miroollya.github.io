Internal View

Web

Flag stored at /internal/flag accessible only from localhost. Direct access returned "Access to localhost is restricted." — bypassed using hexadecimal IP encoding.

BYPASS TECHNIQUE

Converted 127.0.0.1 to hex 0x7f000001 to bypass localhost restriction filters.

Trivial 2

Crypto

Given challenge.txt and challenge.py. Reversed the encryption formula: x = √(E(x) − 1) to recover each character.

Python — Decrypt
import math

encrypted_list = [4490, 7226, 6725, 7057, 5330, 6085, 9026, 4490, 7057, 4901,
                  15130, 9605, 2705, 9802, 11450, 9026, 3026, 2305, 9026, 12997,
                  2305, 2305, 3026, 13226, 15626]

def decrypt(encrypted_list):
    decrypted_chars = []
    for encrypted_value in encrypted_list:
        decrypted_value = int(math.sqrt(encrypted_value - 1))
        decrypted_chars.append(chr(decrypted_value))
    return ''.join(decrypted_chars)

print("Decrypted flag:", decrypt(encrypted_list))

Bluff

Crypto

Given an encrypted flag array. Brute-forced all 4-byte XOR key combinations across the 2-byte key space, checking for the CURTIN_CTF{ prefix.

Python — Brute Force XOR
import itertools

enc_flag = [66, 188, 199, 131, 75, 166, 245, 147, 87, 161, 232, 166,
            114, 223, 228, 181, 118, 222, 226, 239, 115, 209, 241, 235,
            63, 212, 169, 236, 65, 214, 173, 239, 59, 108]

def decrypt(enc_flag, key):
    decrypted_flag = []
    key_len = len(key)
    for i in range(len(enc_flag)):
        chunk = i // key_len
        offset = i % key_len
        decrypted_char = (enc_flag[i] ^ key[offset]) - chunk
        decrypted_flag.append(chr(decrypted_char))
    return ''.join(decrypted_flag)

for key in itertools.product(range(256), repeat=4):
    key_bytes = [int(k) for k in key]
    try:
        flag = decrypt(enc_flag, key_bytes)
        if flag.startswith("CURTIN_CTF{") and flag.endswith("}"):
            print(f"Key: {key_bytes}, Flag: {flag}")
            break
    except:
        continue

09

RSA1

Crypto

Given n, diff_pq, e, and c. Used SageMath to factor n via the quadratic formula, then decrypted with standard RSA.

KEY INSIGHT

With p − q known, solve: p = (diff_pq + √(diff_pq² + 4n)) / 2 using SageMath's arbitrary-precision sqrt.
