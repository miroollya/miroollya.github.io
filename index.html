I'm a Network Security Engineer with a deep passion for cybersecurity, cloud infrastructure, and competitive hacking.   • Zero Trust Tunnel to route traffic securely without exposing origin infrastructure.
  • Zero Trust Architecture (ZTA) to restrict all unauthorized access by default.
